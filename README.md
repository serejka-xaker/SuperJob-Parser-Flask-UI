# Парсер вакансий SuperJob с веб-интерфейсом

## Описание
Это Flask-приложение предназначено для парсинга вакансий с сайта [SuperJob](https://www.superjob.ru/) по заданным профессиям и временным интервалам. Приложение использует OAuth 2.0 для аутентификации API SuperJob и сохраняет результаты в Excel-файл.

## Особенности
- Авторизация через OAuth 2.0.
- Гибкий поиск по ключевым словам и профессиям.
- Извлечение структурированных данных (обязанности, требования, условия).
- Сохранение результатов в формате Excel.
- Веб-интерфейс для запуска процесса парсинга.

## Требования
- Python 3.10+
- Учетная запись разработчика SuperJob API

## Установка
1.  Склонируйте репозиторий:
    ```bash
    git clone https://github.com/serejka-xaker/SuperJob-Parser-Flask-UI.git
    cd SuperJob-Parser-Flask-UI
    ```
2.  Создайте и активируйте виртуальное окружение:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Linux/Mac
    # или
    venv\Scripts\activate     # Windows
    ```
3.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Настройка
1.  Зарегистрируйте приложение в [SuperJob Dev Center](https://api.superjob.ru/register/) для получения `client_id` и `client_secret`.
2.  Создайте файл `Файлы/config.ini` и добавьте туда полученные ключи:
    ```ini
    [Settings]
    client_id = YOUR_CLIENT_ID
    secret_key = YOUR_SECRET_KEY
    ```
    **Важно:** Убедитесь, что `config.ini` добавлен в `.gitignore`.
3.  Убедитесь, что файл `Файлы/Опции.xlsx` находится в нужной директории. Он должен содержать листы с названиями "Профессии", "Кодовые слова", "Требования", "Условия", "Обязанности".

## Запуск
1.  Запустите приложение:
    ```bash
    python app.py
    ```
2.  Откройте браузер и перейдите по адресу `http://127.0.0.1:8000`.

## Структура проекта
- `app.py`: Основной скрипт Flask-приложения.
- `requirements.txt`: Зависимости проекта.
- `Файлы/form.html`: HTML-шаблон для формы ввода дат.
